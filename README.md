# 🎙️ AI Voice Agent 🚀

> An end-to-end **AI Voice Assistant** built with Python, FastAPI, and VAPI that can understand voice, process requests using AI, and respond in real-time with speech.

---

## 🌟 Features

- 🎤 Real-time voice conversations  
- 🧠 AI-powered responses (LLM integration)  
- 🔊 Speech-to-Text & Text-to-Speech  
- 📅 Appointment booking system  
- 💾 Database storage for conversations  
- ⚡ FastAPI backend for high performance  
- 🧪 Dummy frontend for testing  

---

## 🛠️ Tech Stack

- 🐍 Python  
- ⚡ FastAPI  
- 🎧 VAPI (Voice AI)  
- 🗄️ SQLite Database  
- 📦 UV Package Manager  
- 🤖 OpenAI / LLM Integration

# 🚀 Installation

### 📥 Clone the Repository

```bash
git clone https://github.com/DiyalMirza/AI-Voice-Agent.git
cd AI-Voice-Agent
📦 Install Dependencies
uv sync

🔐 Configure Environment Variables
Create a .env file in the root directory and add your API keys:
- OPENAI_API_KEY=your_api_key
- VAPI_API_KEY=your_vapi_api_key

▶️ Run the Application
python backend.py
